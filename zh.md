## 如何运行？
- 从"Releases"下载源码。
- 使用VSCode打开。
- 安装"Vite"插件(非必要)。
- 依次使用终端运行以下命令。
  > cd Seismic-Stations-Viewer<br>
  > npm install -g pnpm<br>
  > pnpm install<br>
  > pnpm run dev<br>

<br>
 
## 如何构建?
- 使用VSCode打开。
- 依次使用终端运行以下命令。
  > cd Seismic-Stations-Viewer<br>
  > pnpm install<br>
  > pnpm build<br>
