## 如何运行？
- 从 Releases 下载代码
- 在 VSCode 中打开
- 安装 “Vite” 扩展。（可选）
- 依次在终端中运行以下命令：
  > `cd Seismic-Stations-Viewer`<br>
  > `npm install -g pnpm`<br>
  > `pnpm install`<br>
  > `pnpm run dev`<br>

 
## 如何构建？
- 在 VSCode 中打开
- 依次在终端中运行以下命令：
  > `cd Seismic-Stations-Viewer`<br>
  > `pnpm install`<br>
  > `pnpm run build`<br>
