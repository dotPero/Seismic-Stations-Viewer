## 如何運行？

- 從 Releases 下載程式碼
- 在 VSCode 中開啟
- 安裝 "Vite" 擴充功能（可選）
- 依序在終端機中執行以下指令：
  > `cd Seismic-Stations-Viewer`<br>  
  > `npm install -g pnpm`<br>  
  > `pnpm install`<br>  
  > `pnpm run dev`<br>

## 如何編譯？

- 在 VSCode 中開啟
- 依序在終端機中執行以下指令：
  > `cd Seismic-Stations-Viewer`<br>  
  > `pnpm install`<br>  
  > `pnpm run build`<br>
